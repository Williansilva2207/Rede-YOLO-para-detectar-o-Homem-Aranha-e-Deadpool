# 🕷️🟥 Detecção de Objetos com YOLO — Homem-Aranha e Deadpool

Projeto de Visão Computacional utilizando o modelo YOLO para detecção de objetos, com foco na identificação dos personagens **Homem-Aranha** e **Deadpool** em imagens.

---

## 📌 Sobre o projeto

Este projeto tem como objetivo aplicar técnicas de **Deep Learning** e **Computer Vision** para treinar um modelo capaz de reconhecer personagens específicos em imagens.

A abordagem utilizada foi baseada no algoritmo **YOLO (You Only Look Once)**, conhecido por sua alta performance em detecção de objetos em tempo real.

---

## 🧠 Como funciona

O modelo YOLO funciona dividindo a imagem em uma grade e prevendo:

* 📦 Bounding boxes (caixas delimitadoras)
* 🎯 Probabilidade de existência de um objeto
* 🏷️ Classe do objeto detectado

Tudo isso acontece em **uma única passagem pela rede neural**, tornando o processo extremamente rápido e eficiente.
